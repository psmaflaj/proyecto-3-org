# Guía de Usuario - Herramientas Hack

Este documento explica cómo ejecutar el Traductor y el Desensamblador desde la consola o terminal.

## Requisitos Previos
- Tener instalado **Python 3.x** en el sistema.
- Abrir la terminal o Símbolo del Sistema (`cmd`) y ubicarse en el directorio donde se encuentran los scripts fuente (`src/`).

---

## 1. Uso del Traductor (Hack Assembler)
Este programa toma un archivo de texto con código ensamblador (`.asm`) y genera un archivo binario ejecutable (`.hack`).

**Comando de ejecución:**
```bash
python HackAssembler.py [nombre_del_archivo.asm]
